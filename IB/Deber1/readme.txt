El presente proyecto consta de tres scripts. 
- index.php
- despliegue.php
- algoritmoDerivada.php 

index.php ==> Se encarga de presentar el problema y solicitar
		al usuario el valor en el que desean evaluar
		la derivada.

despliegue.php ==> Se engarga de pasar los datos de una array
			asociativo a una tabla en html y ademas
			estilza la tabla como tambien hace
			referencia al algoritmo principal. 

algoritmoDerivada.php ==> Contiene la logica principal del 
				proyecto.
